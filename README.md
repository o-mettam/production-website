# Owen Mettam - Professional Portfolio Website

A modern, responsive personal portfolio website built with HTML, TailwindCSS, and JavaScript.

## Features

- **Modern Design**: Clean, professional layout with gradient accents
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth animations, hover effects, and transitions
- **Contact Form**: Functional contact form with validation
- **Skill Bars**: Animated skill proficiency indicators
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Mobile Menu**: Responsive hamburger menu for mobile devices

## Structure

- `index.html` - Main HTML structure with semantic markup
- `script.js` - JavaScript for interactivity and animations
- `README.md` - Documentation

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About**: Personal summary and professional overview
3. **Experience**: Professional work history
4. **Education**: Academic background
5. **Skills**: Technical and professional skills with visual indicators
6. **Contact**: Contact information and message form

## Technologies Used

- HTML5 Semantic Markup
- TailwindCSS for styling
- Vanilla JavaScript for interactivity
- Font Awesome for icons
- Google Fonts (Inter) for typography

## Customization

To customize this website for your own use:

1. Update personal information in `index.html`
2. Replace placeholder content in Experience, Education, and Skills sections
3. Update contact information
4. Modify color scheme by adjusting CSS variables
5. Add your own social media links

## Getting Started

1. Clone or download the files
2. Open `index.html` in a web browser
3. No build process required - it's a static website

## Browser Compatibility

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge

## Mobile Responsiveness

The website is fully responsive and includes:
- Hamburger menu for mobile navigation
- Flexible grid layouts
- Touch-friendly interface
- Optimized typography for smaller screens
